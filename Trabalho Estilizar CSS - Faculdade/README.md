💻 Estilizador de Produtos (CSS)

Projeto desenvolvido para a disciplina de front-end com foco em estruturação HTML e estilização com CSS, simulando um card de produto para e-commerce.

📌 Sobre o projeto

Este projeto consiste na criação de um card de produto moderno, contendo:

Título do produto

Descrição

Preço antigo e atual

Lista de características

Botões de ação

Tudo estilizado com CSS para melhorar a aparência e usabilidade.

🎯 Objetivo

Praticar conceitos fundamentais de:

HTML semântico

CSS (estilização, cores, espaçamento, tipografia)

Organização de layout

Criação de componentes visuais

🛠️ Tecnologias utilizadas

HTML5 | CSS3 | Visual Studio Code

📂 Estrutura do projeto
📁 Trabalho Estilizar CSS - Faculdade
├── index.html
├── style.css

📚 Aprendizados

Durante o desenvolvimento, foram praticados:

Criação de layout com div

Uso de classes CSS

Estilização de botões

Organização de conteúdo visual

📌 Observações

Este é um projeto acadêmico com fins de aprendizado.
